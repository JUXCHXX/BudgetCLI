Database Schema
===============
transactions table:
├── id (PK)
├── type (income | expense)
├── category
├── amount
├── date (ISO)
├── note
└── created_at
   Indices: date, category, type

budgets table:
├── id (PK)
├── category (UNIQUE)
├── monthly_limit
└── created_at
   Index: category

Getting Started
===============
1. git clone https://github.com/yourusername/BudgetCLI.git
2. cd BudgetCLI
3. pip install -e ".[dev]"
4. pytest
5. make all
6. budget init
7. budget transaction add --help
